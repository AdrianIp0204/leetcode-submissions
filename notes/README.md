# Notes

Use this folder for patterns, mistakes, and short explanations that are useful after solving.

- `reflection-template.md` - the standard for problem notes.
- `model-answers/` - curated model solutions and explanation notes to compare against Adrian's actual accepted attempts.
- `attempt-conventions.md` - how to preserve failed and alternative accepted attempts.
- `js-ts-quick-reference.md` - compact JavaScript/TypeScript syntax notes for LeetCode work.
- `next-problems.md` - generated queue for what to practice next.

Run `npm run audit:model-answers` after adding model-answer notes.
